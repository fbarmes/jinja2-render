
# jinja2_render


## Usage

```
usage: jinja2_render.py [-h] [-o OUTFILE] [-e EXTRA_VARS]
                        infile varfiles [varfiles ...]

positional arguments:
  infile                jinja2 template
  varfiles              var file

optional arguments:
  -h, --help            show this help message and exit
  -o OUTFILE, --outfile OUTFILE
                        output file (default: None)
  -e EXTRA_VARS, --extra_vars EXTRA_VARS
                        extra variables (default: None)
```
